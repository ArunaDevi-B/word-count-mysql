To run the Node app follow the bellow instructions and to start the react application check the Readme file in react repo.

1. Clone the repo named 'word-count-mysql' in your local.
2. Once cloned run the command npm install.
3. Add the below line in your package.json "scripts".
 "start": "nodemon index.js",

4. create a .env file and add the below credential details.
port,
host,
user,
password,
database

 5. Give the command npm start to run the node application.
